# History dates

## Запуск проекта

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/NoWizarrd/test_only.git
   cd test_only
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите локальный сервер:
   ```bash
   npm run start
   ```
4. Откройте приложение по адресу [http://localhost:3000](http://localhost:3000)
